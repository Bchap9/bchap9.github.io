Course Overview

Web Application Programing & Hacking - Dr. Phu Phung

General

The Purpose behing the Web Application Programing & Hacking (WAPH) course is to learn about the web development process through experimentation and hands on experiences. Using a combination of lecture, lab, hackathons, and personal projects, we will explore how various element sinteract as well as exploring attack vectors for a website for the purpose of learning to best code against them.

Labs

The labs for this course mainly consist of in lecture activities and small code snipets for the purpose of experimentation. In these labs we focus on one smaller part of the code at a time to break down how each part works together.

Hackathons

The purpose of hackathons is to build our knowledge of how websites can be attacked. By knowing the most common attack vectors for a web page we can make better considerations when developing our web page to prevent a webpage from ever being exploited. The hackathons target things like unvalidated forms to leverage SQL injection or JavaScript Injection.

Projects

The 2 individual and 1 group projects for the course are geared towards the application of the knowledge learned. By using the coding knowledge from the labs/lecture in conjunction with the brest practices learned in the hackathons, we can develop functional web pages.
